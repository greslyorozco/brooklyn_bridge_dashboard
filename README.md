# Brooklyn Bridge Pedestrian Dashboard

Small Streamlit dashboard that explores pedestrian traffic on the Brooklyn Bridge.

The dataset shows pedestrian counts by hour. It includes total pedestrians, people walking toward Manhattan, people walking toward Brooklyn, weather, temperature, and precipitation.

## What This Dashboard Shows

This dashboard lets the user choose between:

1. Hourly traffic
2. Daily traffic
3. Weekly traffic

The dashboard also shows:

1. A line chart of pedestrian traffic
2. Total pedestrian count
3. Average pedestrian count
4. Pedestrians going toward Manhattan
5. Pedestrians going toward Brooklyn
6. A weather filter
